# MiniShop

A simple REST API built with:

- Node.js
- Express.js
- MongoDB
- Mongoose

## Features

- Get all products
- Get product by ID
- Create product
- Update product
- Delete product

## Technologies

- Express
- MongoDB
- Mongoose

## Run

```bash
npm install
npm start
```

MongoDB must be running locally on:

```
mongodb://127.0.0.1:27017/miniShop
```
